# The-logistics-company
# 📦 Logistics Delivery Analysis

## 📊 Project Overview
This project analyzes the delivery process of a logistics company to identify delays, optimize operations, and calculate KPIs.

## 🧱 Data Sources
- Orders dataset
- Campaign dataset
- Process dataset
- Shipping dataset

## ⚙️ Workflow
1. Data audit & exploration
2. Data cleaning (per dataset)
3. Data merging
4. Delivery logic modeling
5. KPI analysis

## 🎯 Goal
Simulate real delivery process and identify bottlenecks.

## 🛠 Tech Stack
- Python (pandas, numpy)
- Jupyter Notebook
- Git

### 📌 Logic Overview
The delivery process follows a rule-based system:
- Orders received on weekends are processed starting Monday
- Warehouse preparation takes 2 days
- Trucks operate on Monday, Wednesday, and Friday
- Express orders skip waiting time
